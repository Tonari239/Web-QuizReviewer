# XAMPP + PHP Server Setup for Web-QuizReviewer

## Prerequisites
- [XAMPP](https://www.apachefriends.org/index.html) installed on your system

## Running the PHP Server
1. Copy or move the `server/` directory into your XAMPP `htdocs` folder (e.g., `C:/xampp/htdocs/Web-QuizReviewer/server`).
2. Start Apache from the XAMPP Control Panel.
3. Access the server in your browser at:  
	`http://localhost/Web-QuizReviewer/server/`

## Project Structure
- `registration-form/` — Frontend HTML/CSS/JS
- `server/` — PHP backend (place your PHP files here)

## Development Tips
- Use the `server/` folder for all PHP scripts and API endpoints.
- You can add more PHP files for different endpoints as needed.
